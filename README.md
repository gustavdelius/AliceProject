# MMath Extended Independent Project (in progress)

## Overview
In this project I propose a multispecies stock-recruitment model that directly incorporates the effects of species interactions at the community level. 
My project will be completed in the following stages:

1. Select data from a relevant group of species from a single ecosystem to use as a [case study](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#1---selecting-a-case-study).
2. Devise a series of differential equations that [model stock-recruitment processes](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#2---devising-the-model) of multiple species at the community level.
3. [Fit the devised model](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#3---fitting-the-model) to the chosen case study data.
4. [Evaluate the success](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#4---evaluating-success) of the model.

I will then produce a written report and additionally give a talk on the results of my work.

## Data
This project uses data from the RAM Legacy stock assessment database (<https://www.ramlegacy.org/>). The folder '[RAM Legacy raw data](https://github.com/alicedwoodley/Extended-Independent-Project/tree/main/RAM%20Legacy%20raw%20data)' contains both the Rdata file and script file describing the contents of the database. 

The file 'Gulf_Of_Alaska.RData' contains the processed data used for all following work. It contains two variables:
- GoA.species : common name, scientific name and RAM Legacy stock ID code for the 6 chosen species.
- GoA.data : spawning stock biomass (SSB) and recruitment time series for all 6 chosen species.
