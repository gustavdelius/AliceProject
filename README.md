# MMath Extended Independent Project (in progress)

## Overview
In this project I propose a multispecies stock-recruitment model that directly incorporates the effects of species interactions at the community level. 
My project will be completed in the following stages:

1. Select data from a relevant group of species from a single ecosystem to use as a [case study](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#1---selecting-a-case-study).
2. Devise a series of differential equations that [model stock-recruitment](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#2---devising-the-model) of multiple species at the community level.
3. [Fit the devised model](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#3---fitting-the-model) to the chosen case study data.
4. [Evaluate the success](https://github.com/alicedwoodley/Extended-Independent-Project/edit/main/README.md#4---evaluating-success) of the model.

I will then produce a written report and additionally give a talk on the results of my work.

## Data
The data used throughout this project is from the RAM Legacy stock assessment database (<https://www.ramlegacy.org/>). The folder '[RAM Legacy R data](https://github.com/alicedwoodley/Extended-Independent-Project/tree/main/RAM%20Legacy%20R%20Data)' contains both the Rdata file and script file describing the contents of the database. 

## Project stages and repository contents

### 0 - Initial research
To improve my understanding of stock-recruitment models and making them in R, I worked through some of chapter 13 of '[Introductory Fisheries Analyses with R](https://derekogle.com/IFAR/)' by Derek Ogle. I made basic plots and fitted single-species stock-recruitment models, first for the Pacific Coast arrowtooth flounder and then for the North Sea European plaice.

### 1 - Selecting a case study (in progress)
The folder '[Case study](https://github.com/alicedwoodley/Extended-Independent-Project/tree/main/Case%20study)' contains the script file in which I search the database for a suitable case study to use. I decide upon the Gulf of Alaska and select a subset of species that are known to interact with each other and have both spawning stock biomass (SSB) and recruitment data recorded for a substantial time period. Plots are then made to give a general overview of the data.
